# Scratch Project Guides

Guides for [Scratch](https://scratch.mit.edu/) projects.

## Contributing

### Global Dependencies

[Node.js](https://nodejs.org/)

### Building and Running

Set Node version

```
nvm use
```

Install project dependencies

```
npm i
```

Run development server

```
npm run dev
```

Build production bundle

```
npm run generate
npm run build
```
